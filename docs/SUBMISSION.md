# 官网填报与提交

## 官方入口

- 中国版权保护中心：https://www.ccopyright.com.cn/
- 著作权登记系统：https://register.ccopyright.com.cn/login.html
- 法规依据：《计算机软件著作权登记办法》：https://www.gov.cn/zhengce/2002-02/20/content_5724627.htm

> 官方页面可能调整，实际填报时以官网当前页面为准。

## 申请流程

1. 打开中国版权保护中心官网，进入著作权登记系统
2. 注册或登录账号，按页面提示完成实名认证
3. 进入软件著作权相关业务，选择**计算机软件著作权登记申请**
4. 在线填写申请表。可打开本工具生成的 `正式资料/申请表信息.txt`，把软件名称、版本号、开发完成日期、开发环境、运行环境、功能说明等内容复制到官网对应字段
5. 上传申请材料——根据官网要求上传 PDF 格式文件和其他证明材料
6. 核对信息无误后提交申请，按官网提示查看受理、补正或登记结果

## 生成文件的用法

| 文件 | 用途 |
|---|---|
| `申请表信息.txt` | 填报辅助，对照填入官网申请表，**不是直接上传的申请材料** |
| `*_操作手册.docx` | 本地编辑稿，确认无误后**导出为 PDF** 上传 |
| `*-代码(前30页).docx` | 本地编辑稿，确认无误后**导出为 PDF** 上传 |
| `*-代码(后30页).docx` | 本地编辑稿，确认无误后**导出为 PDF** 上传 |
| `*-代码(全部).docx` | 不足 60 页时使用，确认无误后**导出为 PDF** 上传 |

DOCX 文件可在 Word、WPS 或 Pages 中继续修改。提交前须另存为 PDF。

## 其他需要准备的材料

以下材料不在本 skill 生成范围内，需自行准备：

- 申请人身份证明（身份证/营业执照复印件）
- 权属证明文件（如涉及权利转移或继承）
- 委托书（如通过代理机构提交）
- 签章页（参考 `软件著作权申请资料/skill/references/signature_page_template.md`）
- 合作开发协议书（如开发方式为"合作开发"，参考 `软件著作权申请资料/skill/references/cooperation_agreement_guide.md`）
