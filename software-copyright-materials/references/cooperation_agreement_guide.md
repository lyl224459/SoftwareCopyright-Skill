# 软件合作开发协议书参考

当软件开发方式为"合作开发"时，著作权登记申请通常需要提交软件合作开发协议书。本参考文件说明该协议书的基本要素，不构成法律建议。

## 适用场景

- 两个或两个以上的自然人、法人或其他组织共同开发软件
- 申请表"开发方式"字段选择"合作开发"

## 协议书基本要素

1. **协议各方基本信息**
   - 各方的名称（或姓名）、地址、联系方式
   - 各方在开发中的角色

2. **合作开发软件的基本信息**
   - 软件名称
   - 版本号
   - 开发起止时间

3. **各方的权利和义务**
   - 各方承担的开发工作内容和范围
   - 各方的技术贡献说明
   - 各方提供的资源（人力、设备、资金等）

4. **著作权归属**
   - 明确著作权的归属方式（共同所有/按份所有）
   - 如果是按份所有，明确各方的权利比例
   - 著作权的行使方式（单独许可/共同许可等）

5. **协议有效期**
   - 协议的生效日期和终止条件

6. **签章**
   - 各方签字或盖章
   - 签署日期

## 注意事项

- 协议书内容应与申请表信息一致（软件名称、版本号、著作权人等）
- 各方签章须真实有效
- 本 skill 不自动生成合作协议书 Docx；合作开发项目的开发者应根据实际情况自行起草或委托法律专业人士起草
- 具体格式和内容以中国版权保护中心官网当前要求为准
