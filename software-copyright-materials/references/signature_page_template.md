# 签章页模板

签章页是软件著作权申请材料中的必备页，用于申请人或代理人签字/盖章。本模板仅供参考，实际使用时请根据当前官网要求调整。

## 模板说明

签章页通常包含以下要素：

1. **软件名称**：与申请表"软件全称"一致
2. **版本号**：与申请表"版本号"一致
3. **著作权人名称**：与申请表"著作权人"一致
4. **申请人签章**：自然人签名或法人盖章
5. **日期**：申请提交日期

## 格式要求

- 签章页应为独立页面，A4 纸张
- 签章信息居中或靠右排列
- 签字/盖章区域留有足够空间
- 如为自然人申请，须手写签名
- 如为法人申请，须加盖公章

## 注意事项

- 签章页上的软件名称和版本号必须与申请表、代码材料页眉、操作手册页眉中的信息完全一致
- 如通过代理机构提交，签章页还须包含代理机构信息
- 具体格式和内容以中国版权保护中心官网当前要求为准
- 本 skill 不自动生成签章页 Docx，用户应根据官网模板准备签章页，确保签章真实有效
