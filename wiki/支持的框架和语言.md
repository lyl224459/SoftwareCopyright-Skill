# 支持的框架和语言

## 项目类型

| 类型 | 描述 |
|---|---|
| Web 前端 | Vue, React, Next.js, Nuxt, Svelte, Astro, Angular 等 |
| Web 后端 | FastAPI, Flask, Django, Gin, Echo, Spring Boot, Express, Laravel, Rails, ASP.NET Core 等 |
| 桌面应用 | Electron, Tauri, PyQt, WPF, WinForms, Qt 等 |
| 移动应用 | Flutter, React Native, MAUI, Xamarin |
| CLI 工具 | Click, Typer, Cobra, Clap, Commander |
| 游戏引擎 | Unity, Godot |
| 机器学习 | TensorFlow, PyTorch, scikit-learn |

## 编程语言

支持 30+ 种语言：TypeScript, JavaScript, Python, Java, Kotlin, Go, Rust, C#, PHP, Ruby, Elixir, Dart, Swift, C, C++, Scala, Groovy, Clojure, F#, Visual Basic, Lua, Zig, Nim, Julia, Haskell, Perl, R, SQL, Shell 等。

## 包管理器

自动检测 npm, pip, Cargo, go mod, Maven, Gradle, pub, Bundler, mix, NuGet, Composer。

完整列表见仓库 [docs/PROJECT_TYPES.md](https://github.com/lyl224459/SoftwareCopyright-Skill/blob/main/docs/PROJECT_TYPES.md)。
