# 快速开始

## 1. 安装依赖

```bash
python3 -m pip install python-docx
```

## 2. 获取 skill

```bash
git clone https://github.com/lyl224459/SoftwareCopyright-Skill.git
```

## 3. 加载到代码助手

**Claude Code：**
```bash
claude --plugin-dir ./SoftwareCopyright-Skill
```

**Codex：**
```bash
mkdir -p ~/.codex/skills
cp -R software-copyright-materials ~/.codex/skills/
```

## 4. 生成资料

在代码助手中打开你的项目，说：

```text
使用 software-copyright-materials 生成当前项目的软件著作权申请资料
```

## 5. 确认和导出

按引导确认信息后，材料生成到 `软件著作权申请资料/正式资料/`。确认无误后将 DOCX 导出为 PDF，上传官网。

## 支持的项目

支持 Web/桌面/CLI/移动/嵌入式等 9 种项目类型，30+ 种语言。

详细说明见 [[支持的框架和语言]]。
