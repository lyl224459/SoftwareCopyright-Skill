# 提交官网指南

## 申请入口

- 中国版权保护中心：https://www.ccopyright.com.cn/
- 著作权登记系统：https://register.ccopyright.com.cn/login.html

## 提交步骤

1. 注册或登录，完成实名认证
2. 进入"计算机软件著作权登记申请"
3. 对照 `申请表信息.txt` 填写在线表单
4. 将 DOCX 导出为 PDF 后上传
5. 上传身份证明等其他材料
6. 核对提交，等待受理

## 需要上传的文件

| 材料 | 来源 |
|---|---|
| 申请表 | 官网在线填写（对照 TXT） |
| 代码鉴别材料 | `*-代码(前30页).pdf` + `*-代码(后30页).pdf` |
| 操作手册 | `*_操作手册.pdf` |
| 身份证明 | 自行准备（身份证/营业执照） |
| 签章页 | 自行准备 |
| 合作协议 | 合作开发时需提供 |

> 官方页面可能调整，以官网当前要求为准。
